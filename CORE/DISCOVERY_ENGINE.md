# DISCOVERY_ENGINE
Version: 1.0
Status: Active

# Purpose

DISCOVERY_ENGINE отвечает за понимание задачи пользователя.

Главная цель — собрать всю информацию, необходимую для проектирования продукта.

Никакие последующие Engines не запускаются до завершения Discovery.

---

# Mission

Понять:

что хочет пользователь;

для кого создается продукт;

какую проблему решает продукт;

какую ценность создает продукт;

какой минимальный результат необходимо получить.

---

# Responsibilities

DISCOVERY_ENGINE обязан:

определить цель проекта;

определить тип проекта;

определить целевую аудиторию;

определить бизнес модель;

определить MVP;

определить ограничения;

определить интеграции;

определить критерии успеха.

---

# Discovery Workflow

Получить описание идеи.

↓

Проверить полноту информации.

↓

Выявить отсутствующие данные.

↓

Сформировать вопросы.

↓

Получить ответы.

↓

Обновить контекст проекта.

↓

Передать проект в PRD_ENGINE.

---

# Required Information

До завершения Discovery должны быть определены:

Project Name

Business Goal

Problem Statement

Target Audience

Primary User

Core Value

MVP Scope

Main Features

Success Metrics

Integrations

Platform

Constraints

---

# Question Rules

Каждый вопрос должен приносить максимум новой информации.

Не задавать вопросы,

ответ на которые можно логически определить.

Не задавать технические вопросы,

если пользователь принимает только бизнес решения.

Если вопрос можно объединить с другим,

объединить.

---

# MVP Rules

Discovery обязан определить минимальный продукт.

Каждая функция проверяется вопросом:

Можно ли выпустить первую версию без нее?

Если да,

функция переносится в Future Scope.

---

# Scope Classification

Все функции распределяются по категориям.

Core

Обязательные.

Recommended

Желательные.

Future

Будущие версии.

---

# Risks

Discovery обязан определить:

технические риски;

бизнес риски;

риски интеграций;

риски масштабирования.

---

# Output

После завершения Discovery формируются:

Discovery Summary

Business Context

Problem Statement

Target Audience

MVP Scope

Project Constraints

Open Questions

Recommendations

Updated Project Context

---

# Completion Criteria

Discovery считается завершенным только если:

цель проекта определена;

MVP определен;

открытых критичных вопросов не осталось;

контекст полностью сформирован.

---

# Failure Rules

Если критически важной информации недостаточно,

Discovery обязан остановиться.

Запуск PRD_ENGINE запрещен.

---

# Final Principle

Главная задача DISCOVERY_ENGINE —

не задавать много вопросов,

а получить максимум понимания проекта при минимальном количестве общения с пользователем.
